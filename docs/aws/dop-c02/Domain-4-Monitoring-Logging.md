## Study Tips

1. **Master CloudWatch** - Custom metrics, metric filters from logs, composite alarms, anomaly detection, dashboards.

2. **Learn log analysis** - CloudWatch Logs Insights query syntax, Athena for S3 log analysis, X-Ray for distributed tracing.

3. **Understand event-driven automation** - EventBridge rules triggering Lambda, SNS notifications, Systems Manager Automation.

4. **Practice metric streams** - Real-time streaming to Kinesis Firehose, S3, third-party destinations.

5. **Study agent configuration** - CloudWatch Agent for custom metrics, SSM Agent for managed instances, unified CloudWatch agent.
