## Study Tips

1. **Master EventBridge** - Event patterns, rules, targets (Lambda, SNS, SQS, Step Functions), event buses, archives.

2. **Learn event-driven architectures** - Fan-out patterns (SNS to multiple SQS), event streaming (Kinesis), dead-letter queues.

3. **Understand automated remediation** - AWS Config rules with auto-remediation, Systems Manager Automation documents, EventBridge triggers.

4. **Practice troubleshooting** - CodePipeline failures, CodeDeploy rollbacks, Auto Scaling issues, container health checks.

5. **Study incident response** - AWS Health dashboards, CloudWatch alarms, X-Ray service maps, OpsCenter for incident management.
