## Domain Overview

**Domain Weight:** 28% of the exam (largest domain)

This domain tests your ability to select and apply appropriate foundation models and AWS AI services for specific business use cases.

**Use Cases:**
- Content creation (articles, marketing copy, product descriptions)

- Report generation

- Amazon Q

## Selecting the Right Service

### Decision Framework

**Use Amazon Bedrock when:**
- Need foundation model access
- Building custom AI applications
- Require fine-tuning capabilities
- Need API-based model access

**Use Amazon Q when:**
- AWS-related questions
- Code development assistance
- Business intelligence needs
- Quick answers without custom development

**Use Amazon Kendra when:**
- Enterprise search requirements
- Document repositories
- Unstructured data search
- High-precision retrieval needed

**Use Amazon Lex when:**
- Need voice/text conversation interface
- Building chatbots
- Call center automation
- Multi-language support needed
